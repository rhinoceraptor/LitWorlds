    scrape.coffee

    scrape is instantiated for a user session, and is used to modify all encore URLs so that the resulting HTML can be AJAXed to the user. The links are modified from http://domain.tld:port/int to #encore/int. This way, the Backbone.js router can handle those links, and the client app will make a call back to the Node.js server for a new page.

    Example usage: s = new scrape(“brn227.brown.wmich.edu”, “7000”, “literaryworlds.me”) s.get_html(‘62’, callback)
